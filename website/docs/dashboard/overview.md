# Overview

Jobly ships with a built-in web dashboard for monitoring and managing jobs.

## Setup

```csharp
app.UseJoblyUI(); // Serves at /jobly
```

## Dashboard

The main dashboard shows real-time statistics, live jobs/sec graph, 24-hour historical chart, and server status.
